**Vynrix** is an experimental **AI-powered startup automation platform** that discovers businesses without websites, generates AI websites automatically, manages outreach, and tracks revenue.

Project-4 represents **Vynrix Android V2**, a **mobile-first Progressive Web App (PWA)** designed to control the entire Vynrix ecosystem.

---

# 🌐 Vision

Vynrix aims to become an **Agentic Operating System for Digital Entrepreneurship** where AI agents can:

• Discover business opportunities  
• Generate websites automatically  
• Contact potential clients  
• Monitor responses  
• Collect payments  
• Scale revenue autonomously  

---

# 🧠 Core Idea

Instead of manually building websites, **Vynrix automates the entire pipeline**.

Business Discovery ↓ AI Website Prototype ↓ Client Outreach ↓ Client Response Monitoring ↓ Full Website Delivery ↓ Payment Collection

All managed from **a single mobile dashboard**.

# 🧩 Features

## Lead Pipeline

Discover and manage potential clients.

- View discovered businesses
- Track pipeline stages
- Generate AI website prototypes

## Agent Monitor

Monitor AI agents running inside the system.

Agents include:

- Lead Scanner
- Prototype Builder
- Outreach Agent
- Monitor Agent
- Payment Agent

# 📦 Installation

Clone the repository:

```bash
git clone https://github.com/Vynrix/Project-4.git

Go to the project folder:

cd Project-4

Install dependencies:

pip install fastapi uvicorn

Run the backend:

uvicorn main:app --reload

Open the frontend:

frontend/index.html


---

📲 Running as an Android App

Because Vynrix is a Progressive Web App, it can be installed directly on Android.

Steps:

1. Open the dashboard in a mobile browser


2. Tap Add to Home Screen


3. Launch it like a native app
